Tutorial
========

This tutorial introduces how to use hatchet, including basics about:

* Using pandas API
* Generating data for hatchet
* Using hatchet's analysis APIs

Hatchet includes standalone Python-based Jupyter notebook examples for this
tutorial. You can find the tutorial notebooks in your hatchet install under
``examples/tutorial/``.
